## Instalação do projeto:

Clonar o projeto:
```bash
  git clone git@github.com:JovemAlex/badge-create.git
```

Entrar na pasta:
```bash
  cd badge-create
```

Abrir a pasta atual no editor de texto (VScode):
```bash
  code .
```

Instalar todas as dependências do projeto:
```bash
  npm i
```

Rodar o projeto em seu navegador:
```bash
  npm start
```
